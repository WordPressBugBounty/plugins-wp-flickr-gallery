Display your Flickr albums, photostream, and photos on any page or widget. Responsive and easy to set up using Flickr API.

== Description ==

This plugin helps you display your Flickr photos and albums directly on your website. Built with PHP API integration and Bootstrap responsive CSS, it works smoothly across all devices and screen sizes.

**View Pro Demo** **[Flickr Gallery Premium](https://awplife.com/demo/flickr-gallery-premium/)** 
**Learn More About Pro:** **[Flickr Gallery Premium Features](https://awplife.com/wordpress-plugins/flickr-gallery-wordpress-plugin/)**
**Where To Buy:** **[Buy Flickr Gallery Premium](https://awplife.com/account/signup/flickr-gallery-premium/)**

You can showcase your Flickr photostream, albums, and profile on any page, post, or sidebar widget using simple shortcodes. The setup process takes just a few minutes.

**Getting Started**

To start using the plugin, you need your **[Flickr User ID](https://awplife.com/how-to-get-your-user-id-of-flickr/)** and **[Flickr API Key](https://awplife.com/how-to-get-your-api-key-of-flickr/)**. Add these in the API Settings page, create your gallery, and paste the shortcode wherever you want it to appear.

**Note:** Flickr currently allows API key creation only for Flickr Pro subscribers.

== Installation ==

1. Install the plugin via WordPress.org plugin directory or upload the files to your server.
2. Activate the plugin through the Plugins menu.
3. Go to Flickr Gallery → API Settings.
4. Enter your Flickr User ID and API Key.
5. Go to Flickr Gallery → Add Flickr Gallery.
6. Configure your gallery settings and save.
7. Copy the shortcode and paste it into any page, post, or widget.

== Frequently Asked Questions ==

= How do I get my Flickr User ID? =

Visit [this guide](https://awplife.com/how-to-get-your-user-id-of-flickr/) that shows you step-by-step how to find your Flickr User ID from your Flickr profile.

= How do I create a Flickr API Key? =

Follow [this tutorial](https://awplife.com/how-to-get-your-api-key-of-flickr/) to create your API key. Note that currently only Flickr Pro subscribers can create API keys.

= Can I display multiple albums on one page? =

Yes, you can create multiple galleries and place their shortcodes on the same page. Each gallery can show a different album from your Flickr account.

= Does this plugin work with Elementor or Gutenberg? =

Yes, the plugin generates shortcodes that work with any page builder including Elementor, Gutenberg, and classic editor.

= Why are my photos not loading? =

Check that your User ID and API Key are entered correctly in the API Settings page. Also verify that your Flickr photos are set to public visibility.

= Can I display private Flickr photos? =

The free version only displays public photos. The Pro version includes support for fetching and displaying private photos.

= How do I add the gallery to a widget? =

Go to Appearance → Widgets, add a Text widget to your sidebar, and paste your gallery shortcode inside it.

= Is this plugin mobile responsive? =

Yes, the gallery is fully responsive and adjusts automatically to different screen sizes including desktops, tablets, and mobile phones.

= What image sizes are supported in the lightbox? =

The plugin supports multiple image sizes for lightbox preview. You can configure the thumbnail and lightbox image sizes in the gallery settings.

= How do I customize the gallery appearance? =

Use the gallery settings to adjust columns, borders, and layout. For advanced customization, the Pro version includes a custom CSS field.

= Where can I get support? =

For questions or issues, please visit the [plugin support forum](https://wordpress.org/support/plugin/wp-flickr-gallery/).
